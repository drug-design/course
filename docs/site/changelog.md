# Changelog

### Nov 17, 2022 
Launch of drugdesign.org the new "drug design" and "drug discovery" sharing platform. The current version includes all "Molecular Conceptor"  content developed by Synergix Ltd between 2001-2013.

### December, 2023 -
I have begun working on rendering molecules in 3D and have chosen the NGL Viewer [nglviewer.org](https://nglviewer.org/) as a powerful is a web-based application for molecular visualization. My first chapter, featuring several 3D molecules, is already complete.
